# Hello, OPPF

A minimal example of an Open-Prompt Project Format project.

## Goal

Provide a single shell script, `hello.sh`, at the project root that prints
exactly:

```
Hello, OPPF!
```

to standard output and exits with status `0`.

## Requirements

- The script must be runnable with `bash hello.sh`.
- It must print the greeting on a single line, followed by a newline.
- It must not print anything else to standard output.
